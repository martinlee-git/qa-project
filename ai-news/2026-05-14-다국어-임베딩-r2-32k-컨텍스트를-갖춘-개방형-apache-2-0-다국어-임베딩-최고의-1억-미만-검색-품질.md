- Hugging Face Enterprise-Ready 설계에 대한 IBM Granite의 블로그 게시물 R1 전체 크기 311M 모델 교육에서 변경된 사항 컴팩트 97M 다국어 모델 구축 벤치마크 결과 다국어 검색 속도 및 처리량 Matryoshka 임베딩(311M) 프레임워크 통합자를 위한 교차 언어 검색 배포 옵션 어떤 모델을 사용해야 합니까?
- 모델 TL;DR 사용해 보기: ModernBERT를 기반으로 구축된 두 가지 새로운 Apache 2.0 다국어 임베딩 모델 — MTEB Multilingual Retrieval(60.3)에서 열려 있는 모든 100M 미만 다국어 임베더를 능가하는 97M 매개변수 컴팩트 모델, MTEB Multilingual Retrieval(500M 매개변수 미만 개방형 모델 중 2위)에서 65.2점을 획득한 311M 전체 크기 모델 마트료시카 지원.
- 둘 다 200개 이상의 언어를 다루고, 52개 언어로 조정되고, 32K 토큰 컨텍스트(64x R1)를 처리하고, 9개 프로그래밍 언어에 대한 코드 검색을 추가합니다.
