Her · Here — '탐정'을 뜻하는 마라티어입니다. Claude Code 세션의 형사입니다. 모든 Claude Code 세션은 모든 회전, 도구 호출 및 토큰이 포함된 .jsonl 파일이라는 추적을 남깁니다. 그러나 실제로는 해당 추적이 쓰기 전용입니다. 에이전트가 프로덕션에 도달한 이유, 컨텍스트 예산이 실제로 어디로 갔는지 또는 실행의 절반을 조용히 소진한 하위 에이전트를 파악하기 위해 4,000줄의 JSON을 읽는 사람은 거의 없습니다. 전제는 간단합니다. 세션 파일을 페이지에 놓고 그녀가 조사하도록 하세요. 그녀는 무슨 일이 일어났는지 평범한 영어로 재구성하고 위험한 움직임(배포, 구성 및 생산 변경, 비밀)을 표시하고 각각의 사건이 발생한 정확한 순서를 추적합니다. 그녀는 토큰이 어디로 갔는지, 어떤 도구, 하위 에이전트, 기술 및 MCP 서버가 사용되었는지, 그리고 명명되고 수정 가능한 패턴이 실행되는 경우에만 Anthropic과 커뮤니티의 모범 사례를 기반으로 더 잘할 수 있었던 일을 보여줍니다. 그녀는 제안하고 결코 주장하지 않으며 말할 가치가 없을 때 침묵을 지킵니다. 부조종사도 내장되어 있습니다. 그녀에게 물어보세요. “이 도구를 사용한 이유는 무엇입니까?”라고 물어보십시오. 그리고 그녀는 추적에서 응답하고, 회전을 인용하고, 정확한 도구 호출을 엽니다. 세션 보기를 위해 하나의 파일을 삭제하십시오. 여러 개를 드롭하여 프로젝트 뷰를 구축하고 한 번에 여러 세션에 걸쳐 질문을 찾아보세요.
