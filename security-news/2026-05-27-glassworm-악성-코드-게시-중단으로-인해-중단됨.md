## Summary
GlassWorm은 2025년부터 GitHub 저장소 300개를 감염시켜 개발자와 조직에 대한 공급망 공격을 가능하게 했습니다. CrowdStrike는 Google 및 Shadowserver Foundation과 협력하여 악성 패키지 및 확장 프로그램을 통해 소프트웨어 개발자를 표적으로 삼는 지속적인 소프트웨어 체인 캠페인인 GlassWorm과 관련된 모든 명령 및 제어(C2) 채널을 동시에 중단한다고 발표했습니다. CrowdStrike는 "적어도 2025년 초부터 GlassWorm 운영자는 소스 코드 저장소, 클라우드 플랫폼, CI/CD 파이프라인 및 패키지 레지스트리에 액세스할 수 있는 인구인 소프트웨어 개발자를 체계적으로 표적으로 삼았습니다."라고 말했습니다. 개발은 개발자가 증가함에 따라 이루어집니다.

CrowdStrike는 Google 및 Shadowserver Foundation과 협력하여 악성 패키지 및 확장 프로그램을 통해 소프트웨어 개발자를 표적으로 삼는 지속적인 소프트웨어 체인 캠페인인 GlassWorm과 관련된 모든 명령 및 제어(C2) 채널을 동시에 중단한다고 발표했습니다. CrowdStrike는 "적어도 2025년 초부터 GlassWorm 운영자는 소스 코드 저장소, 클라우드 플랫폼, CI/CD 파이프라인 및 패키지 레지스트리에 액세스할 수 있는 인구인 소프트웨어 개발자를 체계적으로 표적으로 삼았습니다."라고 말했습니다. 이번 개발은 개발자들이 점점 더 소프트웨어 공급망 공격을 차단하는 수익성 있는 표적이 되어 공격자가 손상된 단일 워크스테이션을 활용하여 수천 개의 다운스트림 조직과 사용자에게 동시에 영향을 미칠 수 있게 됨에 따라 이루어졌습니다. GlassWorm은 작년에 등장한 이후 Microsoft VS Code Marketplace와 Open VSX 모두에 게시된 트로이 목마 VS Code 확장을 사용하여 "다각적인 캠페인"을 수행하여 Cursor, Positron, Windsurf 및 VSCodium과 같은 VS Code 포크 사용자를 표적으로 삼는 것이 가능해졌습니다. 이 캠페인은 손상된 npm 및 Python 패키지를 통해 악성 코드를 도입한 것으로도 알려져 있습니다. 공격의 최종 목표는 자격 증명 수집, 암호화폐 지갑 유출, 시스템 프로파일링 기능을 갖춘 데이터 도난 프레임워크를 제공하는 것입니다.
