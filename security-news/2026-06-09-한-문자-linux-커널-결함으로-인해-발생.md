## Original Description

CVE-2026-23111 is a Linux kernel nf_tables use-after-free that lets an unprivileged local user escalate to root and escape a container.

Exodus Intelligence는 6월 8일에 전체 기술 연습을 발표했으며 이는 최초의 공개 익스플로잇도 아닙니다. FuzzingLabs는 4월에 독립적인 복제물을 게시했습니다. 결함은 단일 이탈 문자, nf_tables의 반전된 검사로 귀결되었으며 업스트림 수정 프로그램은 이를 한 줄로 제거했습니다. 우분투는 결함 CVSS 7.8(높음)을 평가합니다. 배포판의 커널 패키지에 아직 수정 사항이 포함되어 있지 않으면 업데이트하고 재부팅하세요. 도달 가능한 설정은 일반적입니다. nf_tables와 권한 없는 사용자 네임스페이스, 일반 계정이 개인 샌드박스 내에서 루트 역할을 하고 그렇지 않으면 도달할 수 없는 커널 코드에 접근할 수 있게 해주는 Linux 기능입니다. 둘 다 기본적으로 대부분의 데스크탑과 많은 서버 빌드에 제공됩니다. 그 자체로는 원격 벡터가 없습니다. 이는 공격자가 발판을 마련하고 권한이 낮은 셸, 손상된 컨테이너 또는 서비스 계정을 호스트의 루트로 전환한 후 도달하는 버그입니다.
