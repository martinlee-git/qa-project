## Summary
VS Code 1.123에서는 자동 업데이트가 활성화된 경우 확장이 최신 버전으로 자동 업데이트되기 전에 2시간의 지연이 추가됩니다. Microsoft는 소프트웨어 공급망 위협에 대처하기 위해 Visual Studio Code(VS Code)가 IDE(통합 개발 환경)에 대한 확장이 자동으로 최신 버전으로 업데이트되기까지 2시간의 지연을 적용할 것이라고 발표했습니다. 마이크로소프트는 "자동 업데이트가 활성화되면 새 버전이 게시된 지 2시간 후에 자동 업데이트되어 문제가 있거나 잠재적으로 손상된 릴리스에 대한 추가 보호 계층이 추가됩니다"라고 말했습니다. 새로운 기능은 VS Code 1.123부터 사용할 수 있습니다. 거대 기술 기업은 사용자에게 여전히 다음과 같은 옵션이 있다고 지적했습니다.

## Original Description

VS Code 1.123 adds a two-hour delay before extensions auto-update to newer versions when automatic updates are enabled.
