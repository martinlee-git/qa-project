## Summary
144 Mastra npm 패키지에 easy-day-js 악성 코드가 추가되어 개발자 시스템과 CI 실행기가 인포스틸러 위험에 노출되었습니다. Endor Labs, JFrog, SafeDep, Socket 및 StepSecurity의 조사 결과에 따르면 인공 지능(AI) 애플리케이션 구축을 위한 인기 있는 오픈 소스 JavaScript 및 TypeScript 프레임워크인 Mastra 네임스페이스("@mastra/*")와 관련된 최대 144개의 npm 패키지가 코드명 easy-day-js라는 소프트웨어 공급망 공격의 일부로 손상되었습니다. Socket은 "단일 npm 계정(ehindero)이 2026년 6월 17일 짧은 기간 내에 Mastra 범위에 걸쳐 140개 이상의 악성 패키지를 대량 게시했습니다"라고 말했습니다. 감염된 패키지 자체에는 악성 코드가 포함되어 있지 않습니다. 인스타그램

## Excerpt
Endor Labs, JFrog, SafeDep, Socket 및 StepSecurity의 조사 결과에 따르면 인공 지능(AI) 애플리케이션 구축을 위한 인기 있는 오픈 소스 JavaScript 및 TypeScript 프레임워크인 Mastra 네임스페이스("@mastra/*")와 관련된 최대 144개의 npm 패키지가 코드명 easy-day-js라는 소프트웨어 공급망 공격의 일부로 손상되었습니다. Socket은 "단일 npm 계정(ehindero)이 2026년 6월 17일 짧은 기간 내에 Mastra 범위에 걸쳐 140개 이상의 악성 패키지를 대량 게시했습니다"라고 말했습니다. 감염된 패키지 자체에는 악성코드가 포함되어 있지 않습니다. 대신 88분에 걸쳐 자동 게시 캠페인으로 설명된 각 패키지의 종속성 목록에 추가된 "easy-day-js"라는 타사 라이브러리를 통해 도입되었습니다. 분석에서 SafeDep은 "easy-day-js"가 암호화폐를 훔치는 원격 액세스 트로이 목마를 다운로드하고 실행하는 "dayjs" 날짜 라이브러리의 복제품이라고 설명했습니다. JavaScript 라이브러리는 2026년 6월 16일 오전 7시 5분(UTC)에 "sergey2016"이라는 npm 사용자에 의해 깨끗하고 완전한 기능을 갖춘 복사본으로 게시되었으며, 악성 변경 사항은 2026년 6월 17일 오전 1시 1분(UTC)에 도입되었습니다. StepSecurity는 "Mastra는 AI 개발과 클라우드 인프라의 교차점에 있기 때문에 해당 패키지는 최신 소프트웨어 개발에서 가장 민감한 자격 증명을 보유한 환경에 일상적으로 설치됩니다"라고 말했습니다. "이로 인해 Mastra 생태계는 공급망 공격에 대한 매우 높은 가치의 목표가 되었습니다.커스."
