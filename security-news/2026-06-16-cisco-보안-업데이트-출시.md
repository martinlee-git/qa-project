## Summary
Cisco는 제한적으로 악용된 후 Catalyst SD-WAN Manager에서 CVE-2026-20262를 패치했으며 연방 수정 사항은 6월 29일에 출시될 예정입니다. Cisco는 실제로 활발하게 악용되고 있는 Catalyst SD-WAN Manager의 중간 심각도 보안 결함에 대한 보안 업데이트를 출시했습니다. CVE-2026-20262로 추적되는 이 취약점의 CVSS 점수는 10.0점 만점에 6.5점입니다. 시스코는 권고문을 통해 "이전 SD-WAN vManage로 알려진 Cisco Catalyst SD-WAN Manager의 웹 UI 취약점으로 인해 인증된 원격 공격자가 파일을 생성하거나 영향을 받는 시스템의 파일 시스템에 있는 모든 파일을 덮어쓸 수 있습니다"라고 밝혔습니다. 네트워킹 장비 회사는 이 문제는 사용자 공급 장치에 대한 부적절한 검증에서 비롯된다고 덧붙였습니다.

## Excerpt
Cisco는 현재 활발하게 악용되고 있는 Catalyst SD-WAN Manager의 중간 심각도 보안 결함에 대한 보안 업데이트를 발표했습니다. CVE-2026-20262로 추적되는 이 취약점의 CVSS 점수는 10.0점 만점에 6.5점입니다. 시스코는 권고문을 통해 "이전 SD-WAN vManage로 알려진 Cisco Catalyst SD-WAN Manager의 웹 UI 취약점으로 인해 인증된 원격 공격자가 파일을 생성하거나 영향을 받는 시스템의 파일 시스템에 있는 모든 파일을 덮어쓸 수 있습니다"라고 밝혔습니다. 네트워킹 장비 회사는 이 문제는 파일 업로드 프로세스 중 사용자가 제공한 입력에 대한 부적절한 검증에서 비롯된다고 덧붙였습니다. 공격자는 조작된 HTTP 요청을 영향을 받는 API 엔드포인트에 보내 기본 운영 체제에서 파일을 생성하거나 덮어쓰기 위해 이 동작을 악용할 수 있습니다. 이는 결국 무기화되어 뿌리까지 올라갈 수 있습니다. 그러나 악용의 성공 여부는 공격자가 이미 최소한 쓰기 권한이 있는 유효한 자격 증명을 갖고 있는지 여부에 달려 있습니다.
