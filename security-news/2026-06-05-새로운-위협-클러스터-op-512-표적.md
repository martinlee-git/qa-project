## Original Description

A newly identified China-linked threat cluster, OP-512, is targeting Microsoft IIS servers with a custom three-web-shell framework for espionage.
## Key Points

- 사이버 보안 연구원들은 맞춤형 웹 셸 프레임워크를 배포하기 위해 Microsoft 인터넷 정보 서비스(IIS) 서버를 표적으로 삼는 것으로 관찰된 OP-512(여기서 "OP"는 "상대"를 나타냄)라는 이전에 보고되지 않은 위협 클러스터를 발견했습니다.
- ReliaQuest는 간첩 활동에 초점을 맞춘 활동이 중국과 연관되어 있다는 신뢰도를 중간에서 높은 수준으로 평가했습니다.
## Excerpt
사이버 보안 연구원들은 맞춤형 웹 셸 프레임워크를 배포하기 위해 Microsoft 인터넷 정보 서비스(IIS) 서버를 표적으로 삼는 것으로 관찰된 OP-512(여기서 "OP"는 "상대"를 나타냄)라는 이전에 보고되지 않은 위협 클러스터를 발견했습니다. ReliaQuest는 간첩 활동에 초점을 맞춘 활동이 중국과 연관되어 있다는 신뢰도를 중간에서 높은 수준으로 평가했습니다. OP-512는 해커 뉴스(The Hacker News)와 공유한 보고서에서 "OP-512는 부문과 지리가 중국 관련 정보 우선 순위와 일치하는 조직에 대해 손상된 IIS(인터넷 정보 서비스) 웹 서버를 통해 간첩 활동을 수행했을 가능성이 높습니다."라고 밝혔습니다. OP-512와 다른 알려진 중국 관련 공격자 간에 중복되는 부분은 발견되지 않았지만 지난 12개월 동안 IIS 웹 서버를 선정한 것은 CL-STA-0048, DragonRank 및 GhostRedirector에 이어 네 번째 위협 그룹입니다. 지난달 Cisco Talos는 중국어를 사용하는 여러 사이버 범죄 그룹이 BadIIS라는 악성 코드 변종을 공유하여 IIS 서버를 감염시키고 있다고 밝혔습니다. IIS 서버는 또한 남아시아, 동아시아, 동남아시아 전역의 정부 및 국방 부문을 표적으로 삼는 중국이 주도한 새로운 간첩 캠페인의 일환으로 SHADOW-EARTH-053의 표적이 되었습니다.
