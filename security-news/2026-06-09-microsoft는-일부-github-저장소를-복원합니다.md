## Summary
Microsoft는 Miasma 웜이 정보 도용자를 주입하기 위해 오픈 소스 프로젝트 중 73개를 손상시킨 후 GitHub 저장소를 일시적으로 제거했음을 확인했습니다. 마이크로소프트는 최근 오픈 소스 프로젝트 중 73개가 손상되어 코드에 정보 도용자가 주입된 보안 사고에 대응하여 일부 GitHub 리포지토리를 일시적으로 제거했다고 확인했습니다. Microsoft 대변인은 이메일을 통해 The Hacker News에 “우리의 우선순위는 고객과 더 넓은 생태계를 보호하는 것입니다.”라고 말했습니다. "잠재적인 악성 콘텐츠를 조사하면서 일부 저장소를 일시적으로 제거했습니다. 이러한 저장소 중 일부는 검토 후 복원되었지만 다른 저장소는 작업이 계속되는 동안 오프라인 상태로 남아 있을 수 있습니다…

## Original Description

Microsoft confirms it temporarily removed GitHub repos after Miasma worm compromised 73 of its open-source projects to inject an information stealer.

- Microsoft 대변인은 이메일을 통해 The Hacker News에 “우리의 우선순위는 고객과 더 넓은 생태계를 보호하는 것입니다.”라고 말했습니다.
## Excerpt
마이크로소프트는 최근 오픈 소스 프로젝트 중 73개가 손상되어 코드에 정보 도용자가 주입된 보안 사고에 대응하여 일부 GitHub 리포지토리를 일시적으로 제거했다고 확인했습니다. Microsoft 대변인은 이메일을 통해 The Hacker News에 “우리의 우선순위는 고객과 더 넓은 생태계를 보호하는 것입니다.”라고 말했습니다. "잠재적인 악성 콘텐츠를 조사하면서 일부 저장소를 일시적으로 제거했습니다. 이러한 저장소 중 일부는 검토 후 복원되었지만 작업이 계속되는 동안 다른 저장소는 오프라인 상태로 유지될 수 있습니다." "조사의 일환으로 우리는 영향을 받은 저장소에서 콘텐츠를 가져왔을 수 있는 소수의 고객에게 알렸습니다. 우리는 계속 조사할 것이며 고객 조치가 필요한 추가 사항이 확인되면 확립된 지원 채널을 통해 직접 연락할 것입니다." 이번 개발은 Windows 제조업체가 코드명 Miasma라는 지속적인 소프트웨어 공급망 캠페인의 일환으로 손상되었다는 보고에 따라 GitHub에서 호스팅되는 수십 개의 오픈 소스 프로젝트에 대한 액세스를 차단한 지 며칠 만에 이루어졌습니다. 감염된 프로젝트 중에는 Python 패키지인 "durabletask"가 포함되어 있습니다. Python 패키지는 TeamPCP로 알려진 사이버 범죄 그룹에 의해 Linux 시스템용으로 설계된 정보 도용 도구를 전달하기 위해 지난 달 처음으로 손상되었습니다.
