## Excerpt
사이버 보안 연구원들은 잠재적으로 원치 않는 프로그램(PUP) 제품군을 배포하기 위해 새 탭 라이브 배경 화면 추가 기능 역할을 하는 152개의 Google Chrome 확장 프로그램 네트워크를 발견했습니다. 클러스터는 38개의 개별 Chrome 웹 스토어 게시자 계정과 3개의 브랜드 백엔드(tabplugins[.]com, yowgames[.]com 및 chromewallpaper[.]com)로 구성됩니다. 총 105,000번 설치되었습니다. 일부 확장 프로그램의 이름은 아래에 나열되어 있습니다. "모든 목록은 Chrome 웹 스토어에서 사용자 데이터를 수집하거나 사용하지 않을 것이라고 선언하는 반면, 링크된 개인 정보 보호 정책은 그 반대를 인정합니다. 확장 프로그램은 IP 주소, ISP, 클릭 수, 리퍼러를 기록하고 해당 데이터를 Google 애드센스, DoubleClick 및 제3자 광고 파트너와 공유합니다."라고 소켓 보안 연구원 Kush Pandya는 말했습니다. 더욱이, 식별된 확장 프로그램의 하위 클러스터는 설치 및 제거 작업 중에 활성화되는 JavaScript 파일("js/bg.js")에 두 개의 하드 코딩된 URL을 정의합니다. Gook과 같은 검색 엔진의 유기적 검색은 알고리즘에 의해 생성된 검색 엔진 결과 페이지(SERP)의 무료 목록을 나타냅니다. 게재 위치는 관련성, 권위, 검색 엔진 최적화(SEO)와 같은 매개변수를 기반으로 하며 스폰서 결과와는 다릅니다.
