## Excerpt
위협 행위자들은 약 4,000개의 활성 설치가 있는 WordPress 플러그인인 Everest Forms Pro의 심각한 보안 결함을 적극적으로 악용하여 임의 코드를 실행하여 사이트 전체를 손상시키고 있습니다. 문제의 취약점은 CVE-2026-3300(CVSS 점수: 9.8)으로, 1.9.12까지의 모든 플러그인 버전에 영향을 미치는 원격 코드 실행 버그입니다. 이 결함에 대한 패치는 2026년 3월 18일 버전 1.9.13으로 출시되었습니다. "이것은 계산 애드온의 process_filter() 함수가 사용자가 제출한 양식 필드 값을 eval()에 전달하기 전에 적절한 이스케이프 없이 PHP 코드 문자열로 연결하기 때문입니다"라고 Wordfence는 말했습니다. "입력에 적용된 sanitize_text_field() 함수는 작은따옴표나 기타 PHP 코드 컨텍스트 문자를 이스케이프하지 않습니다. 이를 통해 양식이 '복잡한 계산' 기능을 사용할 때 인증되지 않은 공격자가 문자열 유형 양식 필드(텍스트, 이메일, URL, 선택, 라디오)에 조작된 값을 제출하여 서버에 임의의 PHP 코드를 주입하고 실행할 수 있습니다." 이 취약점을 성공적으로 악용하면 인증되지 않은 악의적인 행위자가 서버에서 임의의 PHP 코드를 실행하여 악성 관리자 계정을 생성하고, 웹 셸을 배포하고, 서버에 더 깊이 침투하여 지속적인 발판을 마련할 수 있는 다른 방법을 열 수 있습니다.
