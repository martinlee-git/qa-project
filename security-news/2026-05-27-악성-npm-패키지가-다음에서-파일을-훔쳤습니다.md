## Summary
악성 npm 패키지가 676회 다운로드되어 GitHub 업로드를 통해 Claude AI 파일을 훔쳐 AI 기반 악성 코드 위험이 증가했습니다. 사이버 보안 연구원들이 npm 레지스트리에서 정보 도용 기능이 포함된 새로운 악성 패키지를 발견했습니다. OX Security에 따르면 "mouse5212-super-formatter"라는 패키지는 Anthropic의 Claude 인공 지능(AI) 도구가 백그라운드에서 업로드 및 출력을 처리하는 데 사용하는 전용 디렉터리인 "/mnt/user-data"에서 파일을 업로드하도록 설계되었습니다. 해당 활동의 코드명은 Malware-Slop 입니다. "악성코드를 분석한 결과, 스크립트는 그 자체를 검증하는 내부 '아카이브 배포 동기화' 유틸리티로 제시되는 것으로 나타났습니다.

## Original Description

Malicious npm package downloaded 676 times stole Claude AI files via GitHub uploads, increasing AI-driven malware risks.

## Excerpt
사이버 보안 연구원들이 npm 레지스트리에서 정보 도용 기능이 포함된 새로운 악성 패키지를 발견했습니다. OX Security에 따르면 "mouse5212-super-formatter"라는 패키지는 Anthropic의 Claude 인공 지능(AI) 도구가 백그라운드에서 업로드 및 출력을 처리하는 데 사용하는 전용 디렉터리인 "/mnt/user-data"에서 파일을 업로드하도록 설계되었습니다. 해당 활동의 코드명은 Malware-Slop 입니다. 연구원인 Moshe Siman Tov Bustan과 Nir Zadok은 "악성코드를 분석한 결과 스크립트는 GitHub 저장소를 검증하거나 초기화하고 경량 '네트워크 상태' 스냅샷을 캡처한 다음 로컬 작업 공간 파일을 원격 추적 트리에 구조적으로 동기화하는 내부 '아카이브 배포 동기화' 유틸리티로 자신을 나타내는 것으로 나타났습니다."라고 말했습니다. 그러나 실제로는 설치 후 단계에서 피해자 환경에서 찾은 GitHub 액세스 토큰이나 하드코딩된 토큰을 폴백으로 사용하여 GitHub에 인증하고 대상 저장소가 존재하는지 확인하고 존재하지 않으면 생성한 다음 모든 파일을 위협 행위자가 제어하는 ​​GitHub 계정에 반복적으로 업로드합니다. 도난당한 파일은 운영자가 다양한 도난 세션을 구별하는 데 도움이 되도록 무작위로 이름이 지정된 폴더에 저장됩니다. 또한 이 악성코드는 가짜 "네트워크 연결" 로그를 작성하여 진단 정보를 보내는 것처럼 보이게 하는 동시에 무단 수집 및 원격 전송과 같은 실제 작동 동작을 모호하게 만듭니다.로컬 데이터의
