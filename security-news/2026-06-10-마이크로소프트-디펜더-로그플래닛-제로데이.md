## Summary
RoguePlanet PoC 익스플로잇은 Microsoft Defender 제로데이 경쟁 조건을 대상으로 하며 성공 시 시스템 수준 셸을 생성할 수 있습니다. Chaotic Eclipse(일명 Nightmare-Eclipse)라는 익명의 보안 연구원이 RoguePlanet이라는 또 다른 Microsoft Defender 제로데이에 대한 개념 증명(PoC) 익스플로잇을 공개했습니다. 새로운 GitHub 계정으로 이 익스플로잇을 게시한 연구원은 "이 익스플로잇은 경쟁 조건이므로 성공할지 실패할지 결정됩니다"라고 "MSNightmare"라고 말했습니다. "어떤 기계에서는 100% 성공률을 달성했지만 다른 기계에서는 어려움을 겪었습니다." 익스플로잇이 성공하면 시스템 수준 권한이 있는 셸이 생성되어 공격자에게 다음과 같은 능력을 부여합니다.

## Original Description

A RoguePlanet PoC exploit targets a Microsoft Defender zero-day race condition and can produce a SYSTEM-level shell when successful.

## Excerpt
Chaotic Eclipse(일명 Nightmare-Eclipse)라는 익명의 보안 연구원이 RoguePlanet이라는 또 다른 Microsoft Defender 제로데이에 대한 개념 증명(PoC) 익스플로잇을 공개했습니다. 새로운 GitHub 계정으로 이 익스플로잇을 게시한 연구원은 "이 익스플로잇은 경쟁 조건이므로 성공할지 실패할지 결정됩니다"라고 "MSNightmare"라고 말했습니다. "어떤 기계에서는 100% 성공률을 달성했지만 다른 기계에서는 어려움을 겪었습니다." 공격이 성공하면 시스템 수준 권한이 있는 셸이 생성되어 공격자에게 임의 코드를 실행하거나 승인되지 않은 작업을 수행할 수 있는 능력이 부여됩니다. 연구원은 이 익스플로잇이 2026년 6월 패치 화요일 업데이트가 설치된 Windows 11 및 10 시스템에서 테스트되었으며 이는 이 익스플로잇이 최신 버전의 데스크톱 운영 체제에서 작동한다는 것을 의미합니다. 즉, "표준 사용자는 ISO 이미지를 마운트할 수 없기 때문에" 현재 형태의 Windows Server 인스턴스에서는 익스플로잇이 작동하지 않습니다. Chaotic Eclipse는 Windows Server 설치도 이 결함에 취약하며 이를 작동하려면 익스플로잇을 다시 설계해야 한다고 강조했습니다.
