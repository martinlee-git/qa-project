## Summary
Fortinet, Ivanti 및 SAP는 심각한 결함을 CVSS 10.0까지 패치하여 RCE, 관리자 인수 및 데이터 노출 위험을 줄였습니다. Fortinet, Ivanti 및 SAP는 임의 코드 실행 및 정보 공개를 초래할 수 있는 여러 가지 중요한 보안 취약성을 해결하기 위해 보안 업데이트를 출시했습니다. Fortinet이 패치한 보안 결함은 FortiSandbox, FortiSandbox Cloud 및 FortiSandbox PaaS WEB UI의 명령 주입 취약점과 관련이 있습니다. CVE-2026-25089(CVSS 점수: 9.1)로 추적됩니다. "FortiSandbox, FortiSandbox Cloud 및 FortiSandbox PaaS WEB UI의 OS 명령 취약점[CWE-78]에 사용된 특수 요소를 부적절하게 무력화하면 인증되지 않은 공격이 허용될 수 있습니다…

## Original Description

Fortinet, Ivanti, and SAP patched critical flaws up to CVSS 10.0, reducing RCE, admin takeover, and data exposure risks.

- Fortinet, Ivanti 및 SAP는 임의 코드 실행 및 정보 공개를 초래할 수 있는 여러 가지 중요한 보안 취약성을 해결하기 위해 보안 업데이트를 출시했습니다.
- Fortinet이 패치한 보안 결함은 FortiSandbox, FortiSandbox Cloud 및 FortiSandbox PaaS WEB UI의 명령 주입 취약점과 관련이 있습니다.
## Excerpt
Fortinet, Ivanti 및 SAP는 임의 코드 실행 및 정보 공개를 초래할 수 있는 여러 가지 중요한 보안 취약성을 해결하기 위해 보안 업데이트를 출시했습니다. Fortinet이 패치한 보안 결함은 FortiSandbox, FortiSandbox Cloud 및 FortiSandbox PaaS WEB UI의 명령 주입 취약점과 관련이 있습니다. CVE-2026-25089(CVSS 점수: 9.1)로 추적됩니다. Fortinet은 "FortiSandbox, FortiSandbox Cloud 및 FortiSandbox PaaS WEB UI의 OS 명령 취약점(CWE-78)에 사용된 특수 요소를 부적절하게 무력화하면 인증되지 않은 공격자가 특별히 제작된 HTTP 요청을 통해 승인되지 않은 명령을 실행할 수 있습니다."라고 말했습니다. 이 문제는 다음 제품 및 버전에 영향을 미칩니다. - 화요일에 Ivanti는 Ivanti Sentry(이전의 MobileIron Sentry)에 영향을 미치는 두 가지 중요한 보안 결함에 대한 수정 사항도 발표했습니다.
