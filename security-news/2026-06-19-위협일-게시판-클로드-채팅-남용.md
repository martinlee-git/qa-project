## Excerpt
이번주에는 인터넷이 끊기지 않았습니다. 설계된 대로 정확하게 사용되었는데, 이는 더 나쁩니다. 그늘진 브라우저 추가 기능을 통해 검색이 이루어졌습니다. AI 채팅 링크가 악성코드 전달 경로로 바뀌었습니다. macOS 공격은 메모리에서 실행되었으며 거의 ​​아무것도 남지 않았습니다. 클라우드 에이전트는 공격자가 이를 개방형 셸처럼 취급하기 전까지는 도우미처럼 보였습니다. 노출된 엣지 장비, 오염된 패키지, 현금 택배 사기, 도둑질, 로더, 더 이상 가장할 필요가 없는 피싱을 추가하세요. 여기에 완전한 혼란이 있습니다. Microsoft는 클라이언트-서버 DNS 트래픽을 위해 Windows Server 2025에서 Windows DNS 서버용 DoH(DNS-over-HTTPS)를 일반적으로 사용할 수 있다고 발표했습니다. 회사는 "일반 가용성을 통해 이제 조직은 암호화되고 인증된 클라이언트-리졸버 DNS 트래픽을 기존 온프레미스 DNS 인프라 내에서 직접 배포할 수 있습니다"라고 말했습니다. "목표는 새로운 확인자 아키텍처 없이 개인 정보 보호를 개선하고, 스푸핑 위험을 줄이고, 제로 트러스트 DNS를 발전시키는 것입니다. Windows DNS 서버에서 DoH를 활성화하면 대부분의 기존 DNS 배포와의 호환성을 유지하면서 HTTPS를 통해 지원되는 클라이언트에 암호화된 통신이 도입됩니다. 조직에서는 DoH 클라이언트와 Windows DNS 서버 간의 DoH 트래픽이 TLS를 통해 암호화되고, DNS 쿼리가 HTTPS 요청으로 전송되고, 기존 DNS 기능이 예상대로 계속 작동하며, 암호화된 DNS와 기존 DNS의 혼합 환경이 지원될 것으로 기대할 수 있습니다." 23개의 사기성 Chrome 브라우저 확장 프로그램 클러스터가 발견되었습니다.결과를 제공하기 전에 사용자의 기본 검색 엔진을 은밀하게 무시하고 수익화 미들웨어를 통해 쿼리를 라우팅하는 것입니다.
